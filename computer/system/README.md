# System

This is part of the operating layer.

System-level workspace policies, registries, routing metadata, and indexes live here. Users should not need to browse this folder to find reports, decks, converted documents, or other final work.

Suggested files:

- `agent-registry.md`
- `workspace-map.md`
- `tool-registry.md`
- `automation-plan.md`

Durable user-facing work belongs under `workspace/projects/{project-slug}/`.
