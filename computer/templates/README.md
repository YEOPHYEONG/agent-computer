# Templates

Shared templates live here.

Agent-specific templates can live inside each agent app.

## PPT Templates

- `ppt-content-spec.md`: slide-by-slide semantic outline and source/reference map
- `ppt-design-spec.md`: visual direction, layout roles, and design system
- `ppt-build-plan.md`: prototype, reconstruction, and QA implementation plan
- `ppt-qa-checklist.md`: premium plan-first mode, editability, content fidelity, layout, and render QA checks
- `ppt-reference-sets/`: open, built-in visual reference boards for common deck styles
