# Instagram Growth Analyst Examples

This folder stores public-safe sample data only. Do not add private account exports, direct messages, credentials, or personally identifying audience data.

Files:

- `sample-posts.csv`: synthetic post-level metrics for smoke testing and demos.

Demo:

```bash
node agents/work/instagram-growth-analyst/tools/analyze-instagram-growth.mjs --input agents/work/instagram-growth-analyst/examples/sample-posts.csv --project instagram-growth-demo
```
