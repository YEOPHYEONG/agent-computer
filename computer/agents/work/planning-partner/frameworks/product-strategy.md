# Product Strategy Lens

Use when the idea must become a product, service, app, workflow, or operational system.

## Useful Questions

- What job is the user hiring this for?
- What is the first narrow wedge?
- What is the smallest version that proves value?
- What must be manual before it becomes automated?
- What moment shows product value clearly?

## Concepts To Borrow Carefully

- Jobs To Be Done
- MVP
- wedge strategy
- activation
- retention
- feature vs workflow
- user journey

## Output Moves

- problem statement
- first user segment
- core use case
- MVP scope
- success signal
- product risks
