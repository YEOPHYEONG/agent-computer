# User Psychology Lens

Use when the idea depends on motivation, identity, desire, anxiety, trust, status, or emotional reward.

## Useful Questions

- What does the user want to become by using this?
- What anxiety or frustration makes the problem feel urgent?
- What emotional reward appears after the first successful use?
- What social proof would make this feel credible?
- What would make the user feel seen rather than sold to?

## Concepts To Borrow Carefully

- identity-based behavior
- loss aversion
- social proof
- cognitive ease
- self-efficacy
- status and belonging
- anxiety reduction

## Output Moves

- emotional job
- identity promise
- trust barrier
- motivation map
- language that matches the user's felt problem
