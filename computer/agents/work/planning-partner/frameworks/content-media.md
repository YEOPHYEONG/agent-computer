# Content And Media Lens

Use when the plan involves newsletters, videos, essays, social channels, media products, education, or creator-led distribution.

## Useful Questions

- What editorial promise repeats every week?
- What format can become a recognizable ritual?
- What does the audience return for?
- What is the balance between utility, taste, identity, and entertainment?
- What distribution path fits the format?

## Concepts To Borrow Carefully

- editorial mission
- content pillars
- recurring formats
- media habits
- audience development
- creator trust
- newsletter/community loops

## Output Moves

- editorial thesis
- repeatable format
- publishing rhythm
- audience promise
- distribution loop
- content-to-product path
