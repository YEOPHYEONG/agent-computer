# Visual Asset Maker QA Checklist

## Source Boundary

- [ ] The asset uses approved source material only.
- [ ] Research, strategy, and factual synthesis were handled upstream when needed.
- [ ] Unsupported claims were not added for visual drama.

## Copy Lock

- [ ] Every string intended for the generated image appears in `copy-lock.md`.
- [ ] Korean text, names, numbers, URLs, dates, claims, and CTA text were flagged.
- [ ] Required user approval was obtained before generation.
- [ ] Text inside final images matches the approved copy closely enough for use.

## Format And Channel

- [ ] Dimensions match the selected channel or are documented as custom.
- [ ] Critical text and faces stay inside safe areas.
- [ ] Thumbnail/card assets remain readable at small preview sizes.
- [ ] Crop variants are clearly named.

## Visual Quality

- [ ] Final visual assets were generated with `$imagegen` / built-in `image_gen`.
- [ ] HTML/CSS/SVG/canvas/Sharp/browser rendering was not used as the primary creative generation engine.
- [ ] Any support renderer use was limited to copy overlays, safe-area guides, contact sheets, package assembly, dimension checks, or QA.
- [ ] If `imagegen` was unavailable, the agent stopped instead of substituting a local renderer as the final generator.
- [ ] The asset matches the creative direction.
- [ ] Brand fit is acceptable or limitations are documented.
- [ ] No incoherent text, distorted logos, broken words, or unreadable labels.
- [ ] No inappropriate or misleading imagery.

## Package Quality

- [ ] Generated, variant, and final assets are separated.
- [ ] File names are clear.
- [ ] Prompt pack is saved.
- [ ] Final handoff says no posting/publishing was performed unless explicitly approved.

## Hard Fail Conditions

- [ ] FAIL if final visual assets were primarily generated by HTML/CSS/SVG/canvas/Sharp/browser rendering.
- [ ] FAIL if `$imagegen` / built-in `image_gen` is not recorded in the generation audit for final assets.
- [ ] FAIL if a project-local one-off renderer becomes the main creative engine instead of a support tool.
