# Visual Asset Format Plan

## Format Presets

| Channel | Recommended Size | Notes |
|---|---:|---|
| Instagram square | 1080x1080 | Feed/grid-safe square |
| Instagram portrait/feed | 1080x1350 | More vertical space |
| Instagram story/reels cover | 1080x1920 | Keep critical text away from UI overlays |
| YouTube thumbnail | 1280x720 | High contrast, large readable text |
| X/Twitter card | 1600x900 | Good for launch posts |
| Blog/newsletter/Open Graph | 1200x630 | Link preview friendly |
| GitHub README hero/showcase | 1600x900 or 1920x1080 | Avoid tiny text |
| Poster | 1080x1350 or custom | Platform dependent |
